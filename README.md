# sandbox_for_cpp
* C++用サンドボックス
* 機能のサンプルコードとかを置いている

## コーディングルール
* サンプルを書く機能の単位で ex_〇〇.cpp を作成する。
* ex.h に ex_〇〇 を追加する。
* 上記追加した ex_〇〇 の namespace 内に Exec() を定義し、定義内に実行するサンプルコードを記述する。
* 挙動確認用に Dummy クラス（dummy.ixx）を用意しているので、機能の挙動を確認する際に利用する。
